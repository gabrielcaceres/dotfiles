# dotfiles
Collection of my dotfiles

## Installs

- Install `bash-completion`

Fedora: `dnf install bash-completion`

Homebrew: `brew install bash-completion@2`


## Set up

`make setup` will:

- Add a line to `.bashrc` to `source` the repo script
- Link Emacs init file
- Link `.inputrc`
- Link `.gitignore`
